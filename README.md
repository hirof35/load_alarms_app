
⏰ Auto-Save Alarm Clock
Pythonの tkinter と pygame を活用した、オートセーブ機能付きのデスクトップアラームアプリケーションです。

🌟 概要
シンプルながらも「使いやすさ」と「確実性」を重視した設計です。
設定したアラームは自動的にJSONファイルへ保存されるため、アプリを再起動しても設定をやり直す必要はありません。また、マルチスレッド処理により、アラーム鳴動中も時計の更新や操作が止まることなくスムーズに動作します。

✨ 特徴機能
永続的データ保存:

alarms.json を介して、登録したアラーム時刻を自動保存・自動読み込み。

マルチスレッド再生:

threading モジュールを採用。音声再生中でもGUIが応答不能にならず、スムーズに「停止ボタン」を押せます。

重複防止ロジック:

データ構造に set 型を用いることで、同じ時刻の二重登録を論理的に防止。

直感的なインターフェース:

秒単位で更新されるデジタルクロック表示。

リストボックスによる視覚的なアラーム管理（追加・削除）。

🛠 技術スタック
GUI: tkinter

Audio: pygame.mixer (高精度なループ再生に対応)

Storage: json (軽量なテキストベース保存)

Concurrency: threading (バックグラウンド再生)

🚀 準備と実行
1. 依存ライブラリのインストール
Bash
pip install pygame
2. 音源ファイルの配置
実行ファイルと同じディレクトリに、アラーム音となるファイルを配置してください。

ファイル名: Clock-Alarm05-1(Mid).mp3
(※ファイル名を変える場合は、コード内の play_alarm 関数内を修正してください)

3. アプリの起動
Bash
python alarm_clock.py
🕹 使い方
入力ボックスに HH:MM 形式（例: 07:30）で時刻を入力し、「追加」ボタンを押します。

時刻になるとアラームが鳴り、メッセージが表示されます。

赤色の「音を止める」ボタンを押すと、再生が停止します。

不要になったアラームはリストから選択して「削除」できます。
